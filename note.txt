Câu lệnh khởi tạo local repository
-> git init

Câu lệnh đẩy các file có sự thay đổi lên staging area
- git add .
- git add (tên file)

Kiểm tra các file đang nằm trên staging area
- git status

lệnh tạo ra commit
- git commit -m "message"
lịch sử commit
- git log

câu lệnh đẩy code lên github lần đầu
- git push -u origin (tên nhánh góc dưới bên tay trái vscode)

câu lệnh push code lên git lần 2
- git push
